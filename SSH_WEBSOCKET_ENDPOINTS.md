# SSH WebSocket端点配置

## 端点地址

前端SSH终端现在支持以下两个WebSocket端点，都连接到运行在8002端口的后端服务：

### 1. 实时SSH终端 (`ws://localhost:8002/ws/ssh`)
- **功能**：提供完整的交互式终端功能
- **用途**：实时命令输入和输出、终端大小调整、会话保持、实时错误处理
- **使用场景**：需要持续交互的SSH会话

### 2. 单次命令执行 (`ws://localhost:8002/ws/ssh/execute`)
- **功能**：执行单次命令并实时返回输出
- **用途**：批量命令执行、脚本运行、监控任务
- **使用场景**：只需要执行单个命令并获取结果

## 前端配置

在 `pages/SshTerminal.vue` 文件中，WebSocket连接地址已更新为：

```typescript
// 交互式终端
const wsUrl = `ws://localhost:8002/ws/ssh`

// 单次命令执行
const wsUrl = `ws://localhost:8002/ws/ssh/execute`
```

## 使用说明

1. **交互式终端模式**：
   - 选择"交互式终端"模式
   - 输入SSH连接信息（主机、端口、用户名、密码）
   - 点击"连接"按钮
   - 使用完整的终端功能

2. **单次命令执行模式**：
   - 选择"单次命令执行"模式
   - 输入SSH连接信息和要执行的命令
   - 点击"执行命令"按钮
   - 查看命令执行结果

## 注意事项

- 确保后端WebSocket服务运行在8002端口
- 前端开发服务器运行在3001端口（可访问 http://localhost:3001/ssh）
- 构建后的生产版本也可正常使用
